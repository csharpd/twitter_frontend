Twitter_frontend
================

## Objectives

Re-create the twitter front end using CSS & HTML5.

## Technologies
* CSS
* HTML
* ERB
* Sinatra

## How to run 
```sh
git clone https://github.com/csharpd/twitter_frontend.git
cd twitter_front
bundle
rackup
```

Open your browser and go to [localhost:9292](http://localhost:9292)
